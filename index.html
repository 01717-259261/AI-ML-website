Why This Project Matters

The COVID-19 pandemic demanded smarter predictions in healthcare. Motivated by real-world impact, our project investigates:

  • Which machine learning model best predicts COVID-19 fatality?
  • Which deep learning architecture is most effective in finding similar medical images?

We started with curiosity. We ended with insights.

Methodology

Machine Learning: KNN, Logistic Regression, and Random Forest were evaluated on a dataset of 1,048,575 patient records. Features were cleaned and scaled. Metrics included Accuracy, Precision, Recall, F1-Score, and ROC-AUC.

Deep Learning: GoogLeNet, ResNet-101, and ZFNet were trained and compared using CALTECH-101 images. Feature embeddings were used for Nearest Neighbour search based on cosine similarity.

Balanced Model Comparison

To visualize the impact of class balancing, we created a comparative chart summarizing the key performance metrics of KNN, Logistic Regression, and Random Forest after applying balancing techniques.

Insights:

  • Random Forest: Achieved the highest accuracy and ROC-AUC, indicating strong general performance.
  • Logistic Regression: Delivered the best F1-score, making it effective for handling imbalanced positive cases.
  • KNN: Maintained competitive recall and precision values after undersampling.

Key Findings

  • Random Forest: Most accurate ML model (94.97%) and best R² (0.72).
  • Logistic Regression: Best recall and F1-score (ideal for detecting actual positives).
  • ResNet: Best generalization and accuracy among CNNs.

What’s Next?

Our limitations were mostly hardware-related. Future plans include:

  • Using Grid Search and Cross Validation for tuning models.
  • Testing newer DL architectures like EfficientNet or DenseNet.
  • Deploying the best models into a clinical web dashboard.
